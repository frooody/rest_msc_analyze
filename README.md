# Анализ рынка заведений общественного питания Москвы

## Описание проекта
Исследовательский анализ данных о заведениях общественного питания Москвы за 2022 год. Проект направлен на выявление ключевых факторов, влияющих на рейтинг заведений, анализ географического и ценового распределения точек, а также на формирование практических рекомендаций для предпринимателей при выборе локации, формата и ценовой стратегии нового бизнеса.

## Цели и задачи
- Оценить структуру рынка общепита: распределение по категориям, административным округам и формату (сетевые/несетевые).
- Выявить зависимости между рейтингом заведения и его характеристиками (ценовая категория, локация, количество мест, статус сети).
- Проанализировать влияние удаленности от центра на средний чек.
- Определить топ-15 популярных сетевых брендов и оценить их средний рейтинг.
- Сформировать обоснованные рекомендации для открытия нового заведения.

## Используемые данные
Исходные данные представлены двумя таблицами:
- `rest_info.csv`: идентификатор, название, адрес, административный округ, категория, график работы, рейтинг, признак сетевости, количество посадочных мест.
- `rest_price.csv`: идентификатор, ценовой сегмент, описание среднего чека, числовая оценка среднего чека, числовая оценка цены капучино.

После объединения и очистки итоговый датасет содержит более 8400 записей о заведениях Москвы.

## Ход работы
Проект реализован в виде Jupyter Notebook и состоит из четырех последовательных этапов:

1. **Загрузка и первичный осмотр данных**
   - Импорт датасетов с использованием `pandas`.
   - Оценка объемов, типов данных и доли пропусков. Выявлено отсутствие данных в столбцах `hours` (~6%) и `seats` (~43%).

2. **Предобработка данных**
   - Объединение таблиц по уникальному идентификатору.
   - Оптимизация типов данных (`float32`, `int8`) для снижения потребления памяти.
   - Стратегическая обработка пропусков: замена на медиану по категориям для `seats`, маркер "Неизвестно" для `hours` и `price`.
   - Удаление неявных дубликатов по паре "название + адрес".
   - Инженерия признаков: создан бинарный признак `is_24_7` для обозначения круглосуточных заведений.

3. **Исследовательский анализ данных (EDA)**
   - Визуализация распределений по категориям и административным округам.
   - Анализ доли сетевых заведений в разрезе категорий.
   - Изучение распределения количества посадочных мест, выявление и обработка выбросов (замена значений выше 95-го перцентиля на медиану по категориям).
   - Анализ распределения рейтингов и построение корреляционной матрицы с использованием библиотеки `phik` для работы с категориальными и числовыми признаками.
   - Изучение зависимости среднего чека от географического положения.
   - Выделение топ-15 сетевых заведений по количеству точек и расчет их средних рейтингов.

4. **Формирование выводов и рекомендаций**
   - Систематизация полученных инсайтов.
   - Подготовка бизнес-ориентированных рекомендаций по выбору локации, позиционированию и ценообразованию.

## Ключевые результаты и выводы
- **Структура рынка:** Лидирующие категории — кафе (~28%), рестораны (~24%) и кофейни (~17%).
- **География:** Более 26% всех заведений сосредоточено в Центральном административном округе. В ЦАО наибольшую долю занимают рестораны (~30%).
- **Сетевой бизнес:** Свыше 60% заведений являются несетевыми. Сетевой формат наиболее развит в сегментах булочных (61%), пиццерий (52%) и кофеен (51%). Лидер по количеству точек — сеть "Шоколадница" (120 заведений). В топ-15 полностью отсутствуют бары и пабы.
- **Рейтинг:** Средний рейтинг по рынку составляет ~4.23. Наиболее сильные корреляции с рейтингом выявлены у ценовой категории (0.324) и административного района (0.316).
- **Ценообразование и география:** Наблюдается четкая градация среднего чека по удаленности от центра. Максимальные значения фиксируются в ЦАО и ЗАО (~1000 руб.), минимальные — в ЮВАО (~450 руб.). Разница между самым дорогим и самым дешевым округами превышает 80%.
- **Посадочные места:** После обработки аномалий типичное количество мест варьируется в зависимости от категории, что отражает разную бизнес-модель форматов.

## Рекомендации для бизнеса
- **Выбор локации:** Для премиум-сегмента приоритетны ЦАО и ЗАО. Для формата быстрого питания или демократичных кафе перспективны ЮВАО и СВАО (ниже конкуренция в дорогом сегменте, стабильный спрос).
- **Позиционирование:** Не рекомендуется прямое копирование концепций топ-15 сетей. Целесообразно рассмотреть ниши с низкой представленностью: бары/пабы, этнические кухни, специализированные кофейни, булочные.
- **Ценообразование:** Использовать медианный средний чек по округу как базовый ориентир. В периферийных районах делать акцент на соотношение "цена/качество", в центральных — на уникальность опыта и уровень сервиса.
